Cette page me permet de tester un format de menu que j'utilise sur très peu de projets, à savoir le menu dit "fullscreen". Je teste également différentes formats d'apparition et différentes animations pour un menu fullscreen. Dans ce cas l'apparition se fait pas le haut mais il suffit de modifier le CSS pour que le menu arrive par la droite ou la gauche ou même par le bas.
